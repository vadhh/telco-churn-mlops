# Eksperimen SML - Afridho Tavadhu

## Project Description
Predicting Customer Churn using the Telco dataset. This project implements an MLOps pipeline covering experimentation, preprocessing automation, and CI/CD via GitHub Actions.

## Directory Structure
- `data_raw/`: Contains the initial `data_telco.csv`.
- `preprocessing/`: 
    - `Eksperimen_Afridho_Tavadhu.ipynb`: EDA and prototyping.
    - `automate_Afridho_Tavadhu.py`: Production-ready script.
- `.github/workflows`: CI/CD configuration.
